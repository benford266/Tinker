# WMI-Display.ino 
This is a pressure sensor / display built around the ESP32 OLED DEV board and a 300PSI transducer.

Wiring
| ESP32 | Sensor  |
|--|--|
| 5v | 5v In |
|GND|GND In |
|A14|Sense
